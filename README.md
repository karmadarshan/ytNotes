# ytNotes
It  is an chrome extension , an educational tool used to enhance efficiency over Youtube Education
As we all take screen shots over youtube when attending a lecture over youtube desktop , It becomes hectic to arrange those screenshots 
this extension helps you manage those screenshots 

Implementations
Steps:-
1 Download Zip File
2 Extract all the zip file 
3 Go to chrome extensions 
4 Click manage extension
5 Turn on  Developer's mode
6 Click Load Unpacked
7 Select extracted file 
Done

How to use :
Steps : - 
1 Open any youtube video
2 On the right hand side of the subtitle icon , A new icon ,capture with snapStream is visible whenever you want a 
  screenshot just click it all the screenshots will be collected .
3 By clicking on ytNotes in extensions now that captured shots can be either deleted or added for pdf
4 For pdf click on  export to pdf
